## Stack Tecnológico

- **Frontend:**  
  - React (Vite)
  - CSS Modules

- **Backend:**  
  - Node.js
  - Express
  - PostgreSQL
  - pg (cliente de PostgreSQL para Node.js)
  - dotenv (manejo de variables de entorno)

- **Base de datos:**  
  - PostgreSQL

- **Herramientas de desarrollo:**  
  - Vite (para desarrollo rápido de React)
  - npm/yarn (gestión de dependencias)
